# Quick Start

## Install

```bash
npm i -g fishxcode-cli
```

Or run directly:

```bash
npx fishxcode-cli@latest fishx setup
```

## Init Flow

```bash
fishx login
fishx setup
fishx doctor
```

## Get API Key

1. Open <https://fishxcode.com/console/token>
2. Create a token
3. Copy it and run `fishx login`
