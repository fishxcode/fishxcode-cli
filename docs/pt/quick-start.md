# Início rápido

## Instalação

```bash
npm i -g fishxcode-cli
```

Ou execução direta:

```bash
npx fishxcode-cli@latest fishx setup
```

## Fluxo inicial

```bash
fishx login
fishx setup
fishx doctor
```

## Obter API Key

1. Acesse <https://fishxcode.com/console/token>
2. Crie um token
3. Copie e execute `fishx login`
