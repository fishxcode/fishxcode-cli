# Início rápido

## Instalação

📦 Pacote npm: <https://www.npmjs.com/package/fishxcode-cli>

```bash
npm i -g fishxcode-cli
```

Ou execução direta:

```bash
npx fishxcode-cli@latest setup
```

## Fluxo inicial

```bash
fishx login
fishx setup
fishx doctor
```

## Obter API Key

1. Acesse <https://fishxcode.com/console/token>
2. Crie um token
3. Copie e execute `fishx login`
