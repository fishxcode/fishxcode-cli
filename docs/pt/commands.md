# Comandos

- `fishx login` Salvar API Key localmente
- `fishx whoami` Mostrar status de autenticação
- `fishx logout` Limpar API Key local
- `fishx setup` Configurar ferramentas suportadas
- `fishx doctor` Verificar ambiente e configuração
- `fishx tools` Listar ferramentas e status
- `fishx balance` Consulta de saldo (placeholder)
- `fishx reset` Resetar configuração local e das ferramentas
