# Inicio rápido

## Instalación

```bash
npm i -g fishxcode-cli
```

O ejecución directa:

```bash
npx fishxcode-cli@latest fishx setup
```

## Flujo de inicio

```bash
fishx login
fishx setup
fishx doctor
```

## Obtener API Key

1. Abre <https://fishxcode.com/console/token>
2. Crea un token
3. Cópialo y ejecuta `fishx login`
