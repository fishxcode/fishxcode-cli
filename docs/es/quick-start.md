# Inicio rápido

## Instalación

📦 Paquete npm: <https://www.npmjs.com/package/fishxcode-cli>

```bash
npm i -g fishxcode-cli
```

O ejecución directa:

```bash
npx fishxcode-cli@latest setup
```

## Flujo de inicio

```bash
fishx login
fishx setup
fishx doctor
```

## Obtener API Key

1. Abre <https://fishxcode.com/console/token>
2. Crea un token
3. Cópialo y ejecuta `fishx login`
