# Comandos

- `fishx login` Guardar API Key localmente
- `fishx whoami` Ver estado de autenticación
- `fishx logout` Limpiar API Key local
- `fishx setup` Configurar herramientas soportadas
- `fishx doctor` Revisar entorno y configuración
- `fishx tools` Listar herramientas y estado
- `fishx balance` Consulta de saldo (placeholder)
- `fishx reset` Restablecer configuración local y de herramientas
