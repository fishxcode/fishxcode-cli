# Commandes

- `fishx login` Enregistrer la clé API
- `fishx whoami` Afficher l'état d'authentification
- `fishx logout` Supprimer la clé API locale
- `fishx setup` Configurer les outils supportés
- `fishx doctor` Vérifier l'environnement et la configuration
- `fishx tools` Lister les outils supportés et leur état
- `fishx balance` Commande de solde (placeholder)
- `fishx reset` Réinitialiser les configurations locales et outils
