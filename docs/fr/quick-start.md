# Démarrage rapide

## Installation

```bash
npm i -g fishxcode-cli
```

Ou exécution directe :

```bash
npx fishxcode-cli@latest fishx setup
```

## Flux d'initialisation

```bash
fishx login
fishx setup
fishx doctor
```

## Obtenir une clé API

1. Ouvrez <https://fishxcode.com/console/token>
2. Créez un token
3. Copiez-le puis exécutez `fishx login`
